MySQL to SQLite3
================

A simple Python tool to transfer data from MySQL to SQLite 3

Installation
------------

.. code:: bash

   pip install mysql-to-sqlite3

Basic Usage
-----------

.. code:: bash

   mysql2sqlite -f path/to/foo.sqlite -d foo_db -u foo_user -p
